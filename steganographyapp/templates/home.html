How It Works

The system follows a secure sender-to-receiver workflow.

1
Sender Encodes

Sender selects image and enters secret message.

2
System Secures

Message is encrypted and hidden using edge-based pixel shuffling.

3
Receiver Decodes

Receiver enters Image ID and Secret Key to reveal the hidden message.

4
Attack Protected

Multiple wrong attempts freeze access and alert the original sender.
